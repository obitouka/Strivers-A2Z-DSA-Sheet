# Contribution Guidelines  

### Steps to Contribute  
1. **Fork** the repository  
2. **Clone** your fork locally:  
```bash
git clone https://github.com/your-username/repo.git
```
3. Inside `src/`, create folder: `YourUsername (Language)`.   *Example:* `src/Obitouka (Cpp)/`  
4. Inside your folder, add solutions organized by [**Striver's-A2Z-DSA-Sheet**](https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-course-sheet-2/)  
5. **Ensure**:  Code is clean and related to topic & never modify other's files  
6. **Commit & Push** your changes  
7. Submit a **PR** with small and clear description
